5.2 Lists#

Creating a List#

c = [-45, 6, 0, 72, 1543]

c

[-45, 6, 0, 72, 1543]

Accessing Elements of a List#

c[0]

-45

c[4]

1543

Determining a List’s Length#

len(c)

5

Accessing Elements from the End of the List with Negative Indices#

c[-1]

1543

c[-5]

-45

Indices Must Be Integers or Integer Expressions#

a = 1

b = 2

c[a + b]

72

Lists Are Mutable#

c[4] = 17

c

[-45, 6, 0, 72, 17]

Some Sequences Are Immutable#

s = 'hello'

s[0]

'h'

s[0] = 'H'

---------------------------------------------------------------------------

c[100]

---------------------------------------------------------------------------

Using List Elements in Expressions#

c[0] + c[1] + c[2]

-39

Appending to a List with +=#

a_list = []

for number in range(1, 6):
    a_list += [number]

a_list

[1, 2, 3, 4, 5]

letters = []

letters += 'Python'

letters

['P', 'y', 't', 'h', 'o', 'n']

Concatenating Lists with +#

list1 = [10, 20, 30]

list2 = [40, 50]

concatenated_list = list1 + list2

concatenated_list

[10, 20, 30, 40, 50]

Using for and range to Access List Indices and Values#

for i in range(len(concatenated_list)):  
    print(f'{i}: {concatenated_list[i]}')

0: 10
1: 20
2: 30
3: 40
4: 50

Comparison Operators#

a = [1, 2, 3]

b = [1, 2, 3]

c = [1, 2, 3, 4]

a == b

True

a == c

False

a < c

True

c >= b

True
